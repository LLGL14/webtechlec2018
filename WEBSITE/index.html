WHAT IS HTML?

HTML stands for Hypertext Markup Language, it is language that is used to create a structure for web pages,websites, and web applications. It is also used for businesses to improve the sales and also give publicity to their businesses. This is also a method that is used for the convenience of the businesses customers.

Content Models -the data that would be placed/used in the HTML
Void Models -Elements that do not have content
Grouping Elements -used grouping elements dividing it to a organized structure. tags article, section, nav, header, footer, span, div
Heading -specific format for text. tags h1,h2,h3,h4,h5

HTTP

  • It retrieves hypertext documents and resources on the WWW
  • Developed by W3C and IETF
  • It runs on top of TCP/IP, using tcp 80 (443 for HTTPS)
  • HTTP resources are identified using URIs(URLs)
  • Based on Client-server Architecture
  • Request-response standard protocol
  • statless communication

Internet
